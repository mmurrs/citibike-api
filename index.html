CitiBike Station Finder

Find the nearest Citi Bike station with real-time availability. $0.01 / request

An MPP paid API that returns the closest Citi Bike stations to any location in NYC, including e-bike counts, dock availability, and walking time estimates.

Use it in Claude Code

1

Install AgentCash (one-time setup):

npx agentcash@latest onboard
2

Ask Claude:

"Find me the nearest Citi Bike station to Times Square with e-bikes available"

That's it. AgentCash handles discovery, payment, and the API call automatically.

API Details

Endpoint: GET /nearest

Params: lat (required), lng (required), limit (optional, default 3, max 10)

Payment: MPP via Tempo USDC $0.01

Example Response

{
  "results": [
    {
      "name": "W 42 St & 8 Ave",
      "distance_meters": 180,
      "walk_minutes": 2,
      "ebikes_available": 4,
      "bikes_available": 12,
      "docks_available": 8,
      "lat": 40.7575,
      "lng": -73.9903
    }
  ]
}
