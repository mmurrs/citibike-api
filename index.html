CitiBike Station Finder

Pick up a bike or find a dock. Real-time availability, paid per request. $0.01 / lookup

A Machine Payments Protocol paid API that returns the closest Citi Bike stations to any location in NYC. Pick up a bike or find a dock to park — with real-time availability and walking time estimates.

Use it in Claude Code

1

Install AgentCash (one-time setup):

npx agentcash@latest onboard
2

Ask Claude:

"Find me a Citi Bike near Bedford Ave — I'm heading to SoHo"

"I'm at the WeWork in Flatiron, where can I dock?"

That's it. AgentCash handles discovery, payment, and the API call automatically.

API Details

Pick up a bike: GET /nearest?lat=…&lng=…&limit=3 $0.01

Returns nearest stations with available bikes and e-bikes.

Park a bike: GET /dock?lat=…&lng=…&limit=3 $0.01

Returns nearest stations with available docks.

Params: lat (required), lng (required), limit (optional, default 3, max 10)

Payment: Machine Payments Protocol via Tempo USDC
